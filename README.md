Performance Analysis of Ski Lifts – Chairlift vs Gondola

Simulation comparing efficiency of an 8-person chairlift and a 10-person gondola in transporting skiers. Focus on throughput, waiting time, and operational breaks over a 1-hour period.

Note: The full implementation of this project is kept private. This repository contains the project description and results.

Key Learnings

Simulation modeling: Implemented deterministic and probabilistic parameters, queues, and random arrivals using Python.

Data analysis: Used CSV outputs for EDA, calculating average waiting times, throughput, and remaining queues.

Statistical testing: Applied Shapiro-Wilk and Mann-Whitney tests to compare lift performance under different conditions.

Problem-solving: Learned to model real-world systems with operational constraints, such as group sizes and random breaks.

Visualization & reporting: Summarized results in tables and charts for clear comparison.
